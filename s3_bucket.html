S3 Bucket module

S3_Bucket class for creation, deletion and listing S3 buckets. Also for adding files to specific Bucket. Logs are saved in logs/s3.log

create_S3_bucket(bucket_name, type='model')

Create an S3 bucket with the specified bucket name and type (model or dataset)

Parameters:

bucket_name (str): Name of the S3 bucket to create. type (str): Type of bucket, either “model” or “dataset”. Defaults to “model”. If type is “model”, will save in the .env file as BUCKET_MODEL, otherwise, will save as BUCKET_DATASET

Returns:

bool: True if bucket creation is successful, False otherwise.
